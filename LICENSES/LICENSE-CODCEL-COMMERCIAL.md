Definition of Automated Code Generation System
----------------------------------------------------------------------

For the purposes of this license, an "Automated Code Generation System"
means any software, service, or platform — other than Codcel itself —
that automatically produces source code, applications, services,
interfaces, or other software artifacts from spreadsheets, models,
specifications, schemas, or other non-code inputs, without direct
human authorship of the resulting source code.

Examples include, but are not limited to:
- Spreadsheet-to-code tools
- Model-to-code or configuration-to-code platforms
- Application, API, or service generators offered to third parties
- Software-as-a-service products that generate software artifacts
  for users or customers

----------------------------------------------------------------------
Exclusion for Developer Tools
----------------------------------------------------------------------

An Automated Code Generation System does NOT include developer
assistance tools such as:

- Integrated Development Environments (IDEs)
- Build tools, compilers, or linters
- Code formatters or refactoring tools
- Autocompletion systems
- AI-assisted coding tools (including, but not limited to,
  GitHub Copilot, ChatGPT, or similar technologies)

provided that the resulting software is written, reviewed, and
maintained by an individual developer or development team and is
not automatically generated for third parties as a product or service.
